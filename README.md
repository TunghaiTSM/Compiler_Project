# Compiler Final Project

## System: Arch Linux

## Instructions:
### 1. `make setup`
- 安裝需要的軟體。

### 2. `make build`
- 從yacc, lex檔案生成我們的編譯器。

### 3. `make test`
- 使用`samples/`資料夾下所列的範例程式進行我們產生的compiler測試。

### 3. `make assembly`
- 使用LLVM裡的`clang`及`llc`，生成組合語言。

### 4. 收工，結束這回合。
